    ABOUT

    Ruby was created in the 1990s by Yukihiro "Matz" Matsumoto in Japan. Matz wanted to create a language that was simple and productive while also still being object oriented. Matz wanted to "minimize programmer pain."

    Ruby is used for web development, along with scritping, automation, games, applications, and prototyping. Ruby on Rails framework is used commonly and made the language more popular.

    Ruby remains a very popular language, especially in web development.
